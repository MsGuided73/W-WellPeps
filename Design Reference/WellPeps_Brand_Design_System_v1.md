# **WellPeps Brand Design System**

This document defines the recommended visual design system for the WellPeps website. It preserves the approved logo while introducing a scalable UI palette for web, mobile, and future marketing materials.

## **1\. Brand Colors**

| Element | Color | Usage |
| :---- | :---- | :---- |
| Logo Navy | \#082B59 | Logo, footer, navigation, premium elements |
| Primary Website Blue | \#1565C0 | Primary CTA buttons, links, active states |
| Accent Blue | \#2EA8F7 | Highlights, gradients, icons, emphasis |
| Headline Blue | \#153B73 | H1-H3 headings |
| Soft Wellness Blue | \#EAF5FF | Section backgrounds |
| Light Gray | \#F7F9FC | Alternate page backgrounds |
| White | \#FFFFFF | Primary page background |
| Gold Accent | \#D8A53A | Thin dividers, subtle premium accents |

## **7\. Overall Design Principles**

* Use generous white space and avoid large dark navy panels.  
* Use the medium blue as the primary interactive color throughout the site.  
* Reserve the bright accent blue for highlights rather than primary buttons.  
* Keep gold minimal and elegant.  
* Maintain rounded corners, soft shadows, and consistent spacing with clear hierarchical structure.  
* Add subtle animations where it will enhance User’s experience..  
* Aim for a clean aesthetic inspired by premium healthcare brands rather than traditional hospital websites.
